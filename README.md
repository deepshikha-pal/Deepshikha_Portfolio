# Personal Portfolio Website

## 🌟 Overview

This is my personal portfolio website built using **HTML, CSS, and JavaScript**. The website showcases my skills, projects, achievements, and contact information in a modern and responsive design.

The goal of this portfolio is to present my work, technical skills, and web development journey while providing an easy way for visitors, recruiters, and collaborators to connect with me.

---

## 🚀 Features

* Responsive Design for Mobile, Tablet, and Desktop
* Modern and Interactive User Interface
* About Me Section
* Skills Showcase
* Education Section
* Projects Section
* Contact Information
* Experience Section
* Smooth Navigation
* Clean and Organized Layout
